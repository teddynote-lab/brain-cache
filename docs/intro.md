# Brain Crew

**Brain Crew**는 기업이 AI를 실질적으로 활용할 수 있도록 돕는 AI 전문 기업입니다.

RAG, LLM Agent, 평가 시스템 등 AI 핵심 기술을 연구하고, 이를 기반으로 엔터프라이즈 솔루션을 구축합니다.

## What We Do

### AI 컨설팅 & 솔루션
기업의 데이터와 업무 프로세스에 최적화된 AI 시스템을 설계하고 구축합니다. PoC부터 프로덕션 배포까지 전 과정을 함께합니다.

### 핵심 기술 영역
- **RAG & Retrieval** — 기업 내부 문서 기반 검색 증강 생성
- **LLM Agent** — 복잡한 업무를 자율적으로 수행하는 AI 에이전트
- **Evaluation** — LLM 애플리케이션의 품질을 체계적으로 측정하고 개선
- **Infrastructure** — 모델 서빙, GPU 최적화, 프로덕션 인프라

## Brain Cache

Brain Cache는 Brain Crew의 기술 블로그입니다.

실무에서 축적한 인사이트, 논문 리뷰, 실험 결과, 프로젝트 경험을 정리하고 공유합니다.

| 섹션 | 설명 |
|------|------|
| **Library** | 외부 아티클, 논문 리뷰, 기술 레퍼런스 큐레이션 |
| **Lab** | 직접 실험하고 리서치한 결과물 |
| **Projects** | 프로젝트 회고와 의사결정 기록 |
| **Seminar & Paper** | 세미나 발표자료, 논문 리뷰, 스터디 정리 |
