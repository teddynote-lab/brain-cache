# Tech Stack

Brain Crew 팀이 프로젝트에서 활용하는 주요 기술 스택입니다.

## LLM & AI

| 영역 | 기술 |
|------|------|
| LLM Provider | Claude (Anthropic), GPT (OpenAI), AWS Bedrock |
| Framework | LangChain, LangGraph, LlamaIndex |
| Embedding | OpenAI, Cohere, BGE, multilingual-e5 |
| Vector DB | Pinecone, Weaviate, Milvus, Chroma |
| Serving | vLLM, TGI, SGLang |
| Evaluation | LangSmith, RAGAS, Custom eval framework |

## Backend & Infrastructure

| 영역 | 기술 |
|------|------|
| API | FastAPI, Python |
| Cloud | AWS (EC2, Bedrock, S3, Lambda) |
| Container | Docker, Kubernetes |
| CI/CD | GitHub Actions |
| Monitoring | Grafana, CloudWatch |

## Data & Search

| 영역 | 기술 |

| Document Processing | Unstructured, PyMuPDF |
| OCR | Tesseract, AWS Textract |
